Quadratic Bézier curves are great when we want to bend a line, but they have their limits. We cannot draw a bulky curve or make smooth connections between line segments.

With cubic Bézier, we have not one but two control points. The first control point defines the curve's initial direction, and the second one sets how the curve should reach its endpoint.

Hover over the image to see the control points. Click here for <a href="https://hunormarton.github.io/svg-curves" target="_blank" rel="noopener">an interactive demo</a> of Cubic Béziers (select Cubic Bézier at the top of the page).

<Example>

    fill="none"
    stroke="red"

## Connecting lines

Cubic Béziers are great for making smooth connections between line segments. We can set the first control point to continue the previous line segment and then place the second control point in line with the following line segment.

<Example>

<SVG width="200" height="200" viewBox="-100 -100 200 200">
  <Path
    d="
      M -90,0 
      L -30,-30
      C 30,-60 60,-30 30,30
      L 0,90"
    fill="none"
    stroke="red"
    stroke-width="10"
  />
</SVG>

{/* prettier-ignore */}
```html
<svg 
  width="200"
  height="200"
  viewBox="-100 -100 200 200"
>
  <path 
    d="
      M -90,0

</svg>
```

</Example>

## How to draw a Profile icon with SVG

A simple icon we can create with cubic Béziers is a profile icon. The head is a circle, and the body is a cubic Bézier and a straight line.

In this example, however, we do not use the `L` command to finish the body. Instead, we use the `Z` command to close the path. The `Z` command closes the path by drawing a straight line from the current point to the starting point.

<Example>

<SVG
  width="200"
  height="200"
  viewBox="-100 -100 200 200"
  fill="none"
  stroke="black"
  stroke-width="10"
>
  <Circle cy="-40" r="20" />
  <Path d="M -50,70 C -50,-20 50,-20 50,70 Z" />
</SVG>

{/* prettier-ignore */}
```html
<svg
  width="200"
  height="200"
  viewBox="-100 -100 200 200"
  fill="none"
  stroke="black"
  stroke-width="10"
>
  <circle cy="-40" r="20" />
  <path
    d="
      M -50,70 
      C -50,-20 50,-20 50,70 
      Z"
  />
</svg>
```

</Example>

## How to draw a gift box with SVG

In this example, the ribbon of the gift box uses a cubic Bezier that smoothly continues the previous straight line and then turns back to the direction of the upcoming line.

<Example>

<SVG width="200" height="200" viewBox="-100 -100 200 200">
  <Path
    class="ribbon"
    d="
      M 0 0
      L 30 0
      C 50 0 50 -25 30 -15
      L 0 0"
    fill="none"
    stroke="red"
    stroke-width="4"
  />
</SVG>

{/* prettier-ignore */}
```html
<svg 
  width="200"
  height="200"
  viewBox="-100 -100 200 200"
>
  <path 
    d="
      M 0 0
      L 30 0
      C 50 0 50 -25 30 -15
      L 0 0"
    fill="none"
    stroke="red"
    stroke-width="4"
  />
</svg>
```

</Example>
